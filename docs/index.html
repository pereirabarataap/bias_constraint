Crime-Scene dataset

Features : float - measurements of a specific crime sample

Target label : {0,1} - usefulness of collected sample

Group label : str - crime-scene location identifier

Our purpose is to increase prediction performance by implementing bias-constraints (aka Orthogonality Coefficient) over the group labels during classifier learning
