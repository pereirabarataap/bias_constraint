Crime-Scene dataset

Features : float >>> scene measurements

Target label : 0/1 >>> useless sample

Group label : int >>> crime-scene location

Our purpose is to assess if we can increase prediction performance by implementing bias-constraints over the group labels during classifier learning

Average Leave-one-group-out (LOGO) test performance (no inner validation) across hyperparameter search space.

Only test crime-scenes (the left-out group) with both 0 and 1 target labels are used for testing since AUC is not computable otherwise.

  • Weighted vs Unweighted -> how average AUC is computed across crime scenes, since test crime scenes have different sample size
  • Not-train-on-all vs Train-on-all -> if only crime scenes with both 0 and 1 target label are used during training or not, respectively
